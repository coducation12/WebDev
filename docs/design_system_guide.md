# BORA Logistics 디자인 가이드라인 (Design System Guide)

보라로지스 브랜드의 핵심 가치인 '연결', '정교함', '미래'를 8가지 시각적 언어로 표현한 디자인 가이드입니다. 향후 로고 및 실물 에셋 적용 시 이 가이드라인을 참고하여 일관성을 유지하십시오.

## 🌈 브랜드 컬러 (BORA Brand Palette)
- **Primary**: `#6A0DAD` (BORA Purple) - 권위와 혁신, 물류의 중심.
- **Secondary**: `#011627` (Deep Navy) - 신뢰와 깊이감.
- **Accent**: 각 시안별 특화 컬러 (Emerald, Orange, Blue 등).

---

## 🏛️ 8가지 디자인 시안(Pillars) 상세 가이드

### 1. Ref 1: Modern Minimal (절제와 신뢰)
- **핵심 키워드**: 여백, 명확성, 럭셔리.
- **디자인 가이드**: 장식적인 요소를 배제하고 세리프(Serif) 폰트를 활용하여 물류의 품격을 제고합니다.
- **이미지 권장 무드**: 광활한 공간감, 자연광이 들어오는 현대적 건축물.

### 2. Ref 2: Visual Dynamic (에너지와 임팩트)
- **핵심 키워드**: 속도, 역동성, 강력한 브랜드 컬러.
- **디자인 가이드**: BORA Purple을 배경과 액센트에 적극 활용하며, 사선(Diagonal) 요소를 통해 추진력을 표현합니다.
- **이미지 권장 무드**: 고속도로의 궤적, 역동적으로 움직이는 물류 수단.

### 3. Ref 3: Next-Gen Trend (테크와 미래)
- **핵심 키워드**: 모듈화, 불투명도, 하이테크.
- **디자인 가이드**: 벤토 그리드(Bento Grid) 레이아웃과 글래스모피즘(Glassmorphism)을 사용하여 테크 플랫폼의 인상을 줍니다.
- **이미지 권장 무드**: 데이터 시각화 그래픽, 미래형 하드웨어 디스플레이.

### 4. Ref 4: Standard Logis (정석과 효율)
- **핵심 키워드**: 비즈니스, 표준, 신뢰.
- **디자인 가이드**: 안정적인 그리드 시스템과 기업 중심의 블루/슬레이트 계열 컬러를 통해 B2B 신뢰를 공고히 합니다.
- **이미지 권장 무드**: 대형 컨테이너 터미널, 글로벌 비즈니스 현장.

### 5. Ref 5: Eco-Connectivity (환경과 ESG)
- **핵심 키워드**: 친환경, 순환, 생명력.
- **디자인 가이드**: 에메랄드 톤과 부드러운 곡선 레이아웃을 사용하며, 유기적인 애니메이션을 적용합니다.
- **이미지 권장 무드**: 숲, 녹색 기술이 적용된 산업 장비, 맑은 바다.

### 6. Ref 6: Cinematic Flow (서사와 몰입)
- **핵심 키워드**: 고해상도 미디어, 스토리텔링, 다크 모드.
- **디자인 가이드**: 전체 화면 미디어를 배경으로 활용하며, 스크롤에 따라 장면이 전환되는 서사적 구조를 가집니다.
- **이미지 권장 무드**: 시네마틱한 야경, 장엄한 스케일의 풍경.

### 7. Ref 7: Apple-esque Reveal (디테일과 모션)
- **핵심 키워드**: 정밀함, 세련된 모션, 프리미엄.
- **디자인 가이드**: 스크롤 연동 애니메이션을 통해 정보를 극적으로 공개하며, 픽셀 단위의 정교한 정렬을 유지합니다.
- **이미지 권장 무드**: 극단적인 매크로 샷(Close-up), 알루미늄/유리 재질감.

### 8. Ref 8: Industrial Blueprint (개념과 설계)
- **핵심 키워드**: 엔지니어링, 분석적, 원형.
- **디자인 가이드**: 모눈종이 배경과 제도선(Rulers)을 활용하여 '설계되는 물류'를 시각화합니다.
- **이미지 권장 무드**: 기술 도면, 회로도, 구조적 골격(Skeleton) 비주얼.

## 📸 Media Asset 가이드
모든 시안에 배치된 플레이스홀더(`MediaPlaceholder`)는 향후 다음 사양에 맞춰 교체해 주십시오:
1. **해상도**: 최소 1920x1080 (Hero 섹션 기준).
2. **저장 방식**: 웹 가독성을 고려한 WebP 혹은 최적화된 MP4.
3. **오버레이**: 텍스트 가독성을 위해 배경 이미지에는 `black/40` 수준의 다크 레이어 적용 권장.
